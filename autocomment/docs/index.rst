======================================================
Die **autocomment** Dokumentation
======================================================
Bei diesem Projekt handelt es sich um den ersten Versuch ein Tool zu schreiben
mit dem C++ Source Code gelesen werden kann. In diesem Fall werden Methoden
aus C++ Source Code extrahiert. Die Argumente und der Return Type dieser
Methoden werden durch auslesen einer Beschreibung aus einer Json Datei
automatisch dokumentiert.
